Get objects in a bounding box:

civicapi.com/bos_fire_hydrants?bbox=-71.062520,42.358028,-71.059516,42.360089

Get entire dataset:

civicapi.com/bos_fire_hydrants/all

Get entire dataset as KML:

civicapi.com/bos_fire_hydrants/kml

jQuery examples:

get objects in a bounding box:
 
              $.ajax({ 
                url: "http://civicapi.com/bos_fire_hydrants", 
                dataType: 'jsonp', 
                data: { 
                  "bbox":"-71.062520,42.358028,-71.059516,42.360089" 
                }, 
                success: function(data){ 
                  // manipulate data 
                } 
              });
get available datasets:
 
              $.ajax({ 
                url: "http://civicapi.com/datasets", 
                dataType: 'jsonp', 
                success: function(data){ 
                  // manipulate data 
                } 
              });
